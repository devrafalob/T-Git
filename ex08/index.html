Código

Imagine que há um código X e que esse código foi inserido dentro de um arquivo html
Então, mantendo a formatação necessaria para ele são postos alguns comandos semanticos juntamente com code. Exemplo:

        
            int i = 9;
            i = i + f;
            if(i==10){
                return true;
            }else{return false;
            }

Feito isso, é possível perceber que no comando pre são considerados tanto espaços quando quebras de linha sem código br

Citações

Para as citações existem alguns comandos importantes, sendo eles q para quote, blockquote que é uma movimentação horizontal de uma citaçckão, cite para adicionar uma url a um determinado trecho e abbr para abreviações, exemplo:

Segundo o boletim médico divulgado neste domingo, Lula deu entrada no Hospital Sírio-Libanês da capital com um ferimento corto-contuso em região occipital – ou seja, um corte na região da nuca.
Assim trazendo maior semântica para o texto, todos os objetos são identificados dentro do código HTML
